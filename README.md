# 🚀 Team Task Manager (MERN)
A full-stack web application to manage projects, assign tasks, and track progress with **role-based access control (Admin/Member)**.
---

## 🌐 Live Demo  
👉 https://ttm-abhishek.up.railway.app/

## 📂 GitHub Repository  
👉 https://github.com/cseabhisheks/TTM

---

## 📌 Features

### 🔐 Authentication
- User Signup & Login (JWT-based)
- Secure password hashing

### 👥 Project & Team Management
- Create and manage projects
- Add/remove team members
- Role-based access (Admin / Member)

### ✅ Task Management
- Create, assign, and update tasks
- Task status tracking (Pending, In Progress, Completed)
- Due date & overdue tracking

### 📊 Dashboard
- View all tasks
- Filter by status
- Overdue task highlighting
---

## ⚙️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)
